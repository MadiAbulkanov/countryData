# Приложение для получения информации о странах

## Описание проекта
Это React-приложение, которое позволяет пользователям просматривать список стран и получать подробную информацию о каждой из них. Приложение использует REST Countries API для получения данных о странах. Пользователь может выбрать страну из списка, чтобы увидеть её основные данные, такие как столица, регион, население, а также список стран, с которыми она граничит.

Этот проект размещен на Vercel. Вы можете посетить его по ссылке ниже:

[Ссылка на проект на Vercel] (https://country-data-indol.vercel.app/)
    
## Список использованных технологий

[React] - библиотека для создания пользовательских интерфейсов.

[TypeScript] - язык программирования для добавления статической типизации.

[CSS] - стилизация элементов интерфейса.

[Axios] - для выполнения HTTP-запросов к REST Countries API.

[REST Countries API] - источник данных о странах.
